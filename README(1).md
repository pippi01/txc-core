# TXZ - Proof of Work Cryptocurrency

![TXZ Logo](https://txzhub.com/images/txz-logo-256.png)

## Overview

TXZ is a proof-of-work cryptocurrency built on Scrypt algorithm with a fixed block reward and sustainable tokenomics.

## Key Features

- **Algorithm:** Scrypt (ASIC-friendly)
- **Block Time:** ~2.5 minutes
- **Block Reward:** 20 TXZ (no halving)
- **Max Supply:** 50,000,000 TXZ
- **Tail Emission:** 2% after max supply
- **Premine:** 5,000,000 TXZ (transparent allocation)
